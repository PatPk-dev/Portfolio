# PEERAPAT.K - Premium Glassmorphic Portfolio & Control Center

พอร์ตโฟลิโอสะสมผลงานระดับพรีเมียมของ **Peerapat Kunsubstid** ออกแบบด้วยดีไซน์กระจกสะท้อนแสง (Glassmorphism) สวยงามล้ำสมัย ขับเคลื่อนด้วยฐานข้อมูล **MySQL (XAMPP / Localhost)** พร้อมระบบหลังบ้านและปุ่มวิเคราะห์แสดงลิงก์เว็บไซต์ผลงานสะสมทั้งหมดแบบกลุ่ม

---

## ✨ คุณลักษณะเด่น (Features)
- **Premium Glassmorphic Design**: รูปลักษณ์สวยงามระดับพรีเมียมด้วย HSL Tailwind/CSS, การไล่สีที่หรูหรา และเอฟเฟกต์โฮเวอร์ที่ตอบสนองอย่างรวดเร็ว
- **Database-Driven 100% (MySQL Localhost)**: ทำงานประสานงานโดยตรงกับ XAMPP MySQL ปราศจาก LocalStorage แคช
- **Admin Control Center**: ระบบหลังบ้านสำหรับจัดการผลงานสะสมอย่างง่ายดาย:
  - เพิ่ม/แก้ไขชิ้นงานด้วยตนเอง พร้อม Live Card Preview แบบเรียลไทม์
  - ระบบเปลี่ยนสถานะชิ้นงานอย่างรวดเร็ว (Published เผยแพร่ / Draft แบบร่าง)
  - ระบบลบผลงานสะสม
- **📋 ฟังก์ชันแสดงชื่อเว็บทั้งหมด**: ปุ่มพิเศษระดับพรีเมียมบนแดชบอร์ด พอกดแล้วระบบจะแสดงรายชื่อผลงานสะสมพร้อมลิงก์ทั้งหมดแบบเว้นบรรทัดเรียงลงมาใน Modal กระจกสุดหรู ทำให้ง่ายต่อการคัดลอก (Copy) ไปใช้งานต่อได้ในคลิกเดียว!
- **Bulk URL Scraper Module**: Paste ลิงก์มาหลาย ๆ บรรทัดพร้อมกัน ระบบจะทำการยิง AJAX Concurrency เพื่อดึงข้อมูล Open Graph (ชื่อเว็บ, คำอธิบาย, รูปภาพปก) และบันทึกลง MySQL ทันที

## 🛠️ วิธีการติดตั้งและเริ่มต้นใช้งาน (Getting Started)

### 1. การจัดเตรียมฐานข้อมูล XAMPP MySQL
1. เปิดโปรแกรม **XAMPP Control Panel** และกดปุ่ม **Start** ข้างข้อความ **MySQL**
2. ไปที่เว็บเบราว์เซอร์เปิดลิงก์ [localhost/phpmyadmin](http://localhost/phpmyadmin)
3. สร้างฐานข้อมูลใหม่ชื่อว่า `portfolio` (ใช้ Encoding: `utf8mb4_general_ci` เพื่อรองรับภาษาไทย)
4. นำเข้า (Import) สคริปต์จากไฟล์ `database.sql` ในโฟลเดอร์หลักของโปรเจกต์ เพื่อสร้างโครงสร้างตารางข้อมูล `projects`

### 2. กำหนดค่าตัวแปรสภาพแวดล้อม (.env Setup)
เปิดไฟล์ `.env` ที่อยู่ใน root directory ของโปรเจกต์ และป้อนข้อมูลฐานข้อมูลโลคอลของคุณ (ตามปกติแล้วของ XAMPP จะใช้ User: root และ Password ว่างเปล่า):
```env
DB_HOST=127.0.0.1
DB_PORT=3306
DB_USER=root
DB_PASSWORD=
DB_NAME=portfolio
```

### 3. รันโปรเจกต์ในเครื่องของคุณ (Local Development)
ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้ง Node.js เรียบร้อยแล้ว:
```bash
# 1. ติดตั้ง Dependencies ของระบบ
npm install

# 2. รันระบบเซิร์ฟเวอร์ด้วย Vercel CLI (ในโหมด Dev เพื่อจำลอง Serverless)
npm start
```
ระบบจะเปิดเบราว์เซอร์และแสดงผลที่ลิงก์ `http://localhost:3000` โดยอัตโนมัติ
