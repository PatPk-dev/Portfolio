# PEERAPAT.K - Premium Glassmorphic Portfolio & Control Center (Database-Free)

พอร์ตโฟลิโอสะสมผลงานระดับพรีเมียมของ **Peerapat Kunsubstid** ออกแบบด้วยดีไซน์กระจกสะท้อนแสง (Glassmorphism) สวยงามล้ำสมัย ทำงานในรูปแบบ **Zero-Configuration (Database-Free)** โดยจัดเก็บข้อมูลชิ้นงานทั้งหมดบนเบราว์เซอร์ **LocalStorage (คีย์: `portfolio_projects`) ล้วน 100%** ทำให้คุณสามารถดาวน์โหลด ติดตั้ง และเปิดใช้งานบนเครื่องตัวเอง หรือนำไป Deploy บน Vercel Production ออนไลน์ได้ทันทีโดยไม่ต้องเชื่อมต่อฐานข้อมูลใดๆ!

---

## ✨ คุณลักษณะเด่น (Features)
- **Premium Glassmorphic Design**: รูปลักษณ์สวยงามระดับพรีเมียมด้วย HSL Tailwind/CSS, การไล่สีที่หรูหรา และเอฟเฟกต์โฮเวอร์ที่ตอบสนองอย่างรวดเร็ว
- **Database-Free 100% (Client-Side Storage)**: ทำงานและจัดเก็บข้อมูลโดยตรงผ่านเบราว์เซอร์ LocalStorage ทำงานได้รวดเร็ว ปราศจากคอขวด และปลอดภัยสูง
- **Admin Control Center**: แผงควบคุมระบบหลังบ้านสำหรับจัดการผลงานสะสมอย่างง่ายดาย:
  - เพิ่ม/แก้ไขชิ้นงานด้วยตนเอง พร้อม Live Card Preview แบบเรียลไทม์
  - ระบบเปลี่ยนสถานะชิ้นงานอย่างรวดเร็ว (Published เผยแพร่ / Draft แบบร่าง)
  - ระบบลบผลงานสะสม
- **📋 ฟังก์ชันแสดงชื่อเว็บทั้งหมด**: ปุ่มพิเศษระดับพรีเมียมบนแดชบอร์ด พอกดแล้วระบบจะแสดงรายชื่อผลงานสะสมพร้อมลิงก์ทั้งหมดแบบเว้นบรรทัดเรียงลงมาใน Modal กระจกสุดหรู ทำให้ง่ายต่อการคัดลอก (Copy) ไปใช้งานต่อได้ในคลิกเดียว!
- **Bulk URL Scraper Module**: Paste ลิงก์มาหลาย ๆ บรรทัดพร้อมกัน ระบบจะใช้ API Serverless ดึงข้อมูล Open Graph (ชื่อเว็บ, คำอธิบาย, รูปภาพปก) และบันทึกลง LocalStorage ให้ทันทีโดยไม่ต้องป้อนทีละช่อง

## 🛠️ วิธีการติดตั้งและเริ่มต้นใช้งาน (Getting Started)

### 1. โคลนหรือดาวน์โหลดโปรเจกต์
ดาวน์โหลดโฟลเดอร์โปรเจกต์มาไว้ในเครื่องคอมพิวเตอร์ของคุณ

### 2. รันโปรเจกต์ในเครื่องของคุณ (Local Development)
ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้ง Node.js เรียบร้อยแล้ว (ไม่จำเป็นต้องใช้ XAMPP หรือ MySQL อีกต่อไป):
```bash
# 1. ติดตั้ง Dependencies (ระบบต้องการเฉพาะสำหรับเครื่องมือวิเคราะห์ลิงก์หลังบ้านตอนพัฒนาในโหมด Dev)
npm install

# 2. รันระบบเซิร์ฟเวอร์ด้วย Vercel CLI (เพื่อรันฟังก์ชันดึงค่า URL หลังบ้าน)
npm start
```
ระบบจะเปิดเบราว์เซอร์และแสดงผลที่ลิงก์ `http://localhost:3000` โดยอัตโนมัติ

### 3. การนำขึ้นระบบออนไลน์ (Production Deployment)
คุณสามารถนำโปรเจกต์นี้ขึ้นไปออนไลน์บน **Vercel** ได้ฟรีทันที 100% เพียงแค่เชื่อมโยง GitHub repository แล้วกด Deploy ได้ทันทีโดยไม่ต้องเชื่อมต่อหรือตั้งค่าฐานข้อมูลใดๆ เพิ่มเติม!
